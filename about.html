Welcome to IITG Webmail Notifier

 

This is IITG Webmail Notifier, a small extension developed by Team UniRank

You Have to Add an account on first run.(Only on first run, not everytime.)

Please message your Suggestions/Feedbacks to any developer on the sidebar or mail to nish.gupta01@gmail.com.

Description

 

IITG Webmail notifier checks your webmail account and notifies the number of unread emails.You need to log in to webmail(It does not matter if you close the webmail tab) It checks for mail after each interval set in options.It can hold connection to webmail as long as the browser remains open.

Features:

  • Interactive GUI
  • Auto and Manual check
  • Can stop auto-logout.
